Built with [Lektor](http://getlektor.com)

### To build:
```
git clone https://github.com/alexjj/alexjj.com.git
cd alexjj.com
lektor build
```
### To run:
```
$ lektor server
```
Go to 127.0.0.1:5000 in your browser

Changes to this repo are automatically deployed by [netlify](netlify.com).
